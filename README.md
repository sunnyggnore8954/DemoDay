# 🪖 헬멧 인식 인공지능

## 📌 프로젝트 소개

이 프로젝트는 2024년 SW동행 데모데이에서 진행한 **공유 모빌리티 이용자의 헬멧 착용 여부를 판별하는 인공지능 프로젝트**입니다.

공유 전동킥보드와 같은 개인형 이동장치의 이용자가 증가하면서 관련 사고도 함께 늘어나고 있지만, 실제 이용 과정에서는 헬멧을 착용하지 않는 경우가 많다는 문제에 주목했습니다. 특히 교통사고 발생 시 머리와 얼굴 부상이 큰 비중을 차지한다는 점을 바탕으로, 단순히 헬멧 착용을 권고하는 것을 넘어 착용 여부를 실제로 확인할 수 있는 방법을 구상했습니다. :contentReference[oaicite:0]{index=0}

이에 사용자의 이미지를 분석하여 헬멧을 착용한 경우 `SAFE`, 착용하지 않은 경우 `DANGER`로 판별하는 인공지능을 개발했습니다.

---

## 🎯 프로젝트 목표

이 프로젝트의 최종 목표는 헬멧 인식 인공지능을 공유 모빌리티 대여 과정에 적용하는 것입니다.

사용자가 모빌리티를 대여하기 전에 자신의 이미지를 입력하면 인공지능이 헬멧 착용 여부를 판별합니다. 헬멧을 착용한 경우에는 정상적으로 이용할 수 있고, 착용하지 않은 경우에는 대여를 제한하는 방식으로 구성했습니다.

이를 통해 다음과 같은 결과를 얻고자 했습니다.

- 공유 모빌리티 사용자의 헬멧 착용 유도
- 교통사고 발생 시 머리와 얼굴의 부상 위험 감소
- 헬멧 착용 의무화 제도의 실효성 향상
- 공유 모빌리티 이용 환경의 안전성 개선

초기에는 CCTV를 이용해 미착용자를 실시간으로 탐지하는 방안도 검토했지만, 설치 비용과 운영 범위를 고려하여 공유 모빌리티 업체의 대여 시스템과 연계하는 방식이 더 실현 가능하다고 판단했습니다. :contentReference[oaicite:1]{index=1}

## ⚙️ 프로젝트 진행 과정

프로젝트는 문제 분석과 아이디어 기획을 먼저 진행한 뒤, 이미지 데이터 구축과 인공지능 모델 개발 순서로 진행했습니다.

- 공유 모빌리티 사고와 헬멧 미착용 문제 조사
- 헬멧 인식 인공지능의 적용 방식 기획
- 헬멧 착용 및 미착용 이미지 데이터 수집
- 이미지별 착용 상태 라벨링
- 이미지 분류 인공지능 모델 학습
- 테스트 이미지를 활용한 착용 여부 판별
- 모델 학습 결과와 예측 결과 시각화

학습된 모델에 새로운 이미지를 입력하면 이미지의 특징을 분석한 뒤 헬멧 착용 여부를 판별합니다. 최종 결과는 `SAFE` 또는 `DANGER`로 출력되도록 구성했습니다. :contentReference[oaicite:2]{index=2}

## 🛠 사용 기술

- **Python**: 전체 인공지능 개발 및 데이터 처리
- **TensorFlow / Keras**: 이미지 분류 모델 설계와 학습
- **OpenCV**: 이미지 크기 조정 및 전처리
- **NumPy**: 이미지와 라벨 데이터 배열 처리
- **Scikit-learn**: 학습 데이터와 테스트 데이터 분리
- **Matplotlib / Seaborn**: 학습 결과와 혼동 행렬 시각화
- **Google Colab**: 데이터 분석 및 모델 학습 환경

## 👤 담당 역할

프로젝트에서 **인공지능 개발자** 역할을 담당했습니다. 팀 구성상 고건희와 이선우가 개발 파트를 맡았으며, 기획팀이 정리한 서비스 흐름을 실제 이미지 분류 기능으로 구현했습니다. :contentReference[oaicite:3]{index=3}

주요 담당 내용은 다음과 같습니다.

- 헬멧 착용 및 미착용 이미지 데이터 수집
- 이미지별 클래스 분류와 라벨 데이터 구성
- OpenCV를 활용한 이미지 전처리
- CNN 기반 헬멧 착용 여부 분류 모델 개발
- 학습 데이터와 테스트 데이터를 이용한 모델 평가
- 조명과 촬영 각도에 따른 오분류 원인 확인
- 부족한 환경의 이미지 데이터를 추가하여 성능 보완
- 예측 결과와 모델 성능 시각화

## 🌱 기대 효과

- 이 프로젝트를 통해 인공지능을 단순히 이미지를 분류하는 기능으로 사용하는 것이 아니라, 실제 사회 문제와 서비스에 연결하는 과정을 경험했습니다.
- 헬멧 인식 기능을 공유 모빌리티 대여 시스템과 연계한다면 사용자의 헬멧 착용을 자연스럽게 유도할 수 있습니다. 이를 통해 교통사고 발생 시 부상 위험을 줄이고, 헬멧 착용 관련 법규가 실제 이용 과정에서 지켜질 수 있도록 지원하는 안전 관리 시스템으로 활용할 수 있습니다.
