# 🪖 헬멧 인식 인공지능

이미지를 분석하여 사용자의 헬멧 착용 여부를 `SAFE`와 `DANGER`로 판별하는 인공지능 프로젝트입니다.  
공유 모빌리티 대여 과정에 적용하여 헬멧 미착용자의 이용을 제한하고, 안전한 이용 환경을 조성하는 것을 목표로 했습니다.

## 📌 프로젝트 배경

- 공유 모빌리티 이용 증가와 함께 교통사고 및 안전 문제가 증가함
- 사고 발생 시 머리와 얼굴 부상의 비율이 높아 헬멧 착용이 중요함
- 헬멧 착용 의무화 제도가 존재하지만 실제 착용 여부를 확인하기 어려움
- 대여 전 헬멧 착용 여부를 자동으로 확인하는 시스템을 기획함

## ⚙️ 주요 기능

- 사용자가 입력한 이미지 분석
- 헬멧 착용 여부 판별
- 헬멧 착용 시 `SAFE` 출력
- 헬멧 미착용 시 `DANGER` 출력
- 분류 결과와 모델 성능 시각화

## 🔄 동작 과정

```text
이미지 입력
→ 이미지 분석
→ 헬멧 착용 여부 판별
→ SAFE 또는 DANGER 출력
→ 미착용 시 모빌리티 이용 제한
